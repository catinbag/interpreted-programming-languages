установить express

```
npm install express
```

установить nodemon

```
npm install -D nodemon
```

удалить пакет

```
npm remove express
```

запустить скрипт dev

```
npm run dev
```

- node index.js - запуск файла
- nodemon index.js - запуск с автоматической перезагрузкой
- npm run dev - вызов псевдонима с помощью npm
